◇自己PR
学生時代、私は自分が未経験のプログラムに挑戦し、技術を向上させることに積極的に取り
組んでいました。その中でも、初めての2Dシューティングゲームの制作に取り組んだ際には、
CSVファイルを使って敵の情報を読み込む処理を行ったり、２Ｄアクションゲームの制作では、
先輩が挑戦していたワイヤーアクションの処理に興味を持ち、苦手な物理計算に挑戦して
実装する事が出来ました。私は制作期間内に出来る限り多くの新しい処理を取り入れる事を
目指していました。また、授業外でも新しい事に挑戦するように意識していました。最近は、
エフェクトやゲーム演出に興味を持ち、ユーザーに見やすく満足してもらえるような演出作りを
心掛けています。今後は、自分の技術を向上させるためだけではなく、将来的には企業が求める
ような最新の技術やユーザーにとってより価値のあるサービスを提供できるように、技術力を更に
向上させていきたいと思っています。
